# ble_scan
-------------------------------------------------------------------------------

## 说明

该示例构建了一个客户端程序，扫描环境中的低功耗蓝牙广播信号。支持被动扫描和主动扫描。该示例程序需要调用 bt_smartbridge 框架。

## 使用方法

在串口终端上输入命令：blescan 命令来启动扫描，其中 blescan passive 表示被动扫描，
